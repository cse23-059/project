Troubleshooting

Solutions to common issues you might encounter while using the application.

Login Problems

  • Forgot password: Use the password reset link on the login page.
  • Account locked: Wait or contact support if you've had multiple failed attempts.
  • Wrong credentials: Check for caps lock and ensure you're using the correct email.

Performance Issues

  • Slow loading: Check your internet connection and try closing other applications.
